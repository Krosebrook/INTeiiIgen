## TL;DR
Generate unit + integration tests.

## Inputs
- `code_under_test`
- `test_framework` (Jest/Vitest/Pytest)

## Output Contract
1. Test files (ready to run)
2. What's tested, what's not
3. How to run (local + CI)

## Example
```typescript
describe('login', () => {
  it('returns token on valid creds', async () => {
    const result = await login('user@example.com', 'pass');
    expect(result.success).toBe(true);
  });
  
  it('returns error on invalid creds', async () => {
    const result = await login('bad@example.com', 'wrong');
    expect(result.success).toBe(false);
  });
});
```
